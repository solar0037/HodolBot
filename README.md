## 실행

먼저 디스코드 API에서 봇 토큰을 가져와서 token.txt에 넣어야 합니다.

Linux

~~~Bash
./scripts/run.sh
~~~

Windows

~~~Powershell
.\scripts\run.ps1
~~~
